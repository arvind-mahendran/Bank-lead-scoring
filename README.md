# Bank Lead Scoring

This repository contains code and artifacts for a lead-scoring project for a bank. The structure follows a standard ML project layout.

Structure

bank-lead-scoring/

- README.md
- requirements.txt
- .gitignore
- data/ (raw and processed subfolders, kept empty or with small samples)
- notebooks/01_baseline_to_profit_threshold.ipynb
- src/ (package with data prep, training, evaluation, prediction)
- models/ (saved model artifacts)
- reports/ (figures and model_card.md)
- app/ (optional Streamlit app)

See src/ for starter modules and comments on implementation.
